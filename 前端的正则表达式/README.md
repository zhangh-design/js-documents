. 用来匹配除了换行符以外的任何单个字符。

[^] ^ 符号轻松获得否定字符集。它会匹配方括号中未包含的所有内容。

i：忽略大小写 （使用这个标志，搜索时不区分大小写）

g：全局匹配 （多亏了这个标志，所有匹配项都能够被找到。如果没有它，将会在找到第一个匹配项后停止。）

* 匹配 0 次或更多次。它实际上等效于 {0,}

? {0,1} （问号可以表示重复前面内容的0次或一次，也就是要么不出现，要么出现一次。）

+ {1,} （表示该表达式可能匹配一次或多次。这类似于星号，但在这里必须至少匹配一次。）
